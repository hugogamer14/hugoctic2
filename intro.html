Esta es una alternativa para poner texto en cursiva pero en este caso lo vamos a poner en negrita, para que se vea más
Por ejemplo: balón.
el procedimiento es el mismo pero en vez de usar "em" usamos "strong"

Encabezados de h1 a h6

Nos sirve para hacer saltos de lineas que permiten que los textos no abunden
tanto y tener más espacio o tambien se utiliza para canciones o poemas

Estilo del texto

Nos sirve para cambiar la forma del texto y también con esto destacar distintas palabras. Se hace utilizando las siglas "em" y poniendo la terminacion hasta donde quieras "/em"

Lista desordenada

Sirve para hacer una lista para cuando no necesitas que algo vaya antes que lo otro, es decir, sin seguir un orden.
Para ello escribimos <"ul">, escribes lo queres colocar con el comando (<"li">) y se cierra con lo siguiente <"/ul">.

Lista ordenada

Sirve igual que la lista desordenada, pero esta utiliza un orden (1. 2. 3. etc.)
Este comando es similar al anterior pero tenemos que escribir en vez de "ul" escribimos "ol"

Viñetas

Sirve para redactar ideas o cosas de forma ordenada o desordenada.
Ejemplos:

Tabla de 1º División

  • Real Madrid
  • Real Sociedad
  • Cádiz

Paises más grandes

  1. Rusia
  2. China
  3. EE.UU
